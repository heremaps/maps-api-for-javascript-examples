Map with Isoline Route for the EV vehicle

Request a range for the EV vehicle with the given parameters and display it on the map

This example calculates the range for the EV vehicle starting its trip from the Brandenburg Gate in the centre of Berlin.

Code

Access to the routing service is obtained from the H.service.Platform by calling getRoutingService(null, 8). The calculateIsoline() method is used to calculate the range for the vehicle with the given parameters as defined in Isoline Routing API. The styling and display of the response is under the control of the developer.
