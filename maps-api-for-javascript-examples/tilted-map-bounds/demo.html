View bounds of the tilted map

Synchronise a static map with an interactive map

This example shows two maps. The upper map allows for panning, zooming, tilting and rotation (to tilt and rotate hold alt key plus left mouse button). The lower map is oriented top-down north-up, it observes the upper map's lookAt position and visualises the visible area of the interactive map.

Visible area visualisation

Code

The following example creates two objects H.map.Polygon and H.map.Marker that represent the current center and the current view bounds of the interactive map. These objects are updated on every mapviewchange event and are marked as volatile at the construction time, that enables the smooth animation of the objects.
