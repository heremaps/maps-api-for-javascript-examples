Map with Driving Route from A to B

Request a driving route from A to B and display it on the map

This example calculates the fastest car route from the Brandenburg Gate in the centre of Berlin (52.51605°N, 13.37787°E) to Friedrichstraße Railway Station (52.52058°N, 13.38615°E), and displays it on the map.

Code

Access to the routing service is obtained from the H.service.Platform by calling getRoutingService(). The calculateRoute() method is used to calculate the fastest car route by passing in the relevant parameters as defined in Routing API. The styling and display of the response is under the control of the developer.
