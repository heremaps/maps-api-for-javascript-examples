Change map style at runtime

Change a style of the map feature when map is instantiated

This example shows how to change map feature style during the map runtime. The map is loaded with the default "normal.map" style, after that the color of the features "landuse.park" and "landuse.builtup" is changed to the dark green and the dark grey accordingly.

Code

The code below adds a change event listener to the instence of the H.map.Style associated with the layer's provider. Within this listener the section of the map style is extracted and modified and merged back to the original style. That results in the changed colors for the "landuse.park" and "landuse.builtup" map features.
