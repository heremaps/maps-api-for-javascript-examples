Polyline on the Map

Display a map highlighting a region or area

This example displays a line connecting the cities of Dublin (53.3477°N, 6.2597°W), London (51.5008°N, 0.1224°W), Paris (48.8567°N, 2.3508°E) and Berlin (52.5166°N, 13.3833°E) and displays it on the map.

Code

A polyline is created using the H.map.Polyline class, passing in an H.geo.LineString holding the points of the line. The look-and-feel of the polyline can be altered by adding the style parameter
