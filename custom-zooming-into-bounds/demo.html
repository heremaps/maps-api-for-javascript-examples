Zoom into Bounds

Zoom into bounds limiting maximum level

This example displays circles on the map, which denote cities' population. By clicking on each circle you can zoom into it, but with different maximum zoom levels, depending on the circle size.

Code

Instead of using setViewBounds method to zoom into circle's bounds, we are using getCameraDataForBounds to compute the best zoom level and map centre for the bounds and then perform a custom processing on it to limit maximum zoom level.
