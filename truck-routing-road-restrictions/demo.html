Map with the truck routes and road restrictions

Show a truck routes with the truck related road restrictions highlighted on the map

This example calculates and displays 2 different truck routes with the same start and end positions, but different truck attributes.
The start position is in Manhattan and the end position on the opposite bank in Newport.

  • The blue route is a simple truck route (without specifying any truck attributes).
  • The fuchsia route shows that a truck with four axles and one trailer transporting flammable goods should take a much longer detour (uses vehicle[axleCount], vehicle[trailerCount], vehicle[hazardousGoods] parameters of the Routing API).

More information about above mentioned routing request parameters can be found here.

The truck restrictions layer is set as a base map.

Code

Access to the routing service is obtained from the H.service.Platform by calling getRoutingService(null, 8). The calculateRoute() method is used to calculate the fastest truck route by passing in the relevant parameters as defined in Routing API. The styling and display of the response is under the control of the developer.
