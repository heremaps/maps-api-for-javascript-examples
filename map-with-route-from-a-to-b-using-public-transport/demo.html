Map with Route from A to B using Public Transport

Request a route from A to B using public transport and display it on the map.

This example calculates a route from the Fernsehturm at Panoramastraße in Berlin (52.5208°N, 13.4093°E) to Kurfürstendamm (52.5034°N, 13.3295°E) in the west of the city using public transport routing, and displays it on the map. The calculation finds the fastest available route using public transport.

Code

Access to the routing service is obtained from the H.service.Platform by calling getRoutingService(). The calculateRoute() method is used to calculate the fastest public transport (mode:fastest;publicTransport) route by passing in the relevant parameters as defined in Routing API. The styling and display of the response is under the control of the developer.
