Search for a Landmark

Request the location of a landmark and display it on the map.

This example makes a landmark search and retrieves the name and location of the Eiffel Tower based on the input text. A clickable marker is placed on the location found.

Code

Access to the geocoding service is obtained from the H.service.Platform by calling getGeocodingService(). The search() method is used to find a landmark by passing in the relevant parameters as defined in Geocoder API. The styling and display of the response is under the control of the developer.
