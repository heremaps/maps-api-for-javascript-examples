Map with postcodes from the Advanced Data Sets

Display a map with the postcode boundaries overlay

This example displays a movable map initially centered on the Brandenburg Gate in the centre of Berlin (52.5159°N, 13.3777°E) with the postcodes overlay provided by the Fleet Telematics Advanced Data Sets REST API. It's possible to display additional information about the postcode area by clicking on the marker.

Code

The example uses H.service.extension.platformData.TileProvider class to fetch data from the Fleet Telematics Advanced Data Sets service. Two providers are configured with the desired geometry types (H.service.extension.platformData.TileProvider.ResultType.POLYLINE for postcode borders and H.service.extension.platformData.TileProvider.ResultType.MARKER for postcode centroids). The result is displayed on the map as two additional layers using an instance of the H.map.layer.TileLayer and H.map.layer.MarkerTileLayer correspondingly.
