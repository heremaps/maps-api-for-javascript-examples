Markers with altitude

Display markers at different altitudes.

This example shows the bicyle route at the Yungas Road (also known as the "Death road") in Bolivia. Markers with the altitude values help to visualise the 3500m of descent for this road.

Code

Access to the routing service is obtained from the H.service.Platform by calling getRoutingService(). The calculateRoute() method is used to calculate the bicycle route by passing in the relevant parameters as defined in Routing API. The markers are supplied with the latitude, longitude and altitude obtained from the routing response, the altitude value provides the elevation for the marker.
