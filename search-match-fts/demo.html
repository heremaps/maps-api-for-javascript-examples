Data Matching with the Advanced Data Sets

Match routing result with the Advanced Data Set.

This example displays a movable map that contains the route polyline and the data about the location of electric vehicle charging stations provided by the Fleet Telematics Advanced Data Sets REST API. The search for the nearest charging stations is made by the polyline's bounding box. Stations that are adjacent to the route itself are marked with green.

Code

The example uses H.service.extension.platformData.SearchRequest class to fetch data from the Advanced Data Sets service. The data event listener matches the provided array of LINK_IDs from the routing service and search results from the Advanced Data Sets layer to apply correct colour to the H.map.Marker.
